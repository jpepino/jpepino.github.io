About SightReader

This web application was built by Jeremy Pepino as part of an Educational Technology semester-long assignment. Jeremy is studying at the Georgia Institute of Technology, pursuing a M.S. in Computer Science through the OMSCS program.

Jeremy is currently working on a DoD contract full-time during the days, and completing the majority of his classwork during the evenings.

Jeremy would like to acknowledge the following entities, through which the completion of SightReader was made possible:

  • VexFlow API: A completely open-source JavaScript API for rendering musical notation. All of the graphics you see on this website were rendered using VexFlow.
  • Nick Kaye's Angular Tutorial: The page layout, design, and navigation were built on top of Nick Kaye's Angular tutorial and template.
  • Piano FF: The audio effects which can be heard in the game interface were downloaded from FreeSound, which were made available by a user called jobro. Thanks jobro!
